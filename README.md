# College Admissions Guide

A website with useful information for students preparing to apply to universities in the US and Europe.

## What's inside
- Differences between IELTS, TOEFL, and Duolingo English Test
- What SAT, ACT, and AP are
- What GPA means, plus a calculator that converts a 5-point grade average into US GPA and a European equivalent

## Why I built this
When I first tried to understand the difference between IELTS, TOEFL, and Duolingo on my own, I couldn't make sense of it. Fortunately, I eventually figured it out — and decided to build this site so no one else would have to struggle with the same confusion.

## Tech stack
- HTML
- CSS
- JavaScript (GPA calculator)

## How to run
Just open `index.html` in your browser — no installation needed.
